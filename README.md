## Projects

### AjayAI — Chat Assistant
An AI-powered chat assistant I built using HTML, CSS, and JavaScript. It connects to the Groq API running Llama 3.3 and gives real-time AI responses in a ChatGPT-style interface. Includes chat history saved in localStorage, typing indicators, markdown formatting, and a mobile-responsive sidebar. Deployed live on GitHub Pages.

🔗 [ajaythakur2923.github.io/ai-chat-assistant](https://ajaythakur2923.github.io/ai-chat-assistant)

### QR Code-Based Attendance System (Capstone)
My Master's capstone project. A cloud-integrated backend built with Python and Flask that processes real-time QR scan events and stores data in a cloud-hosted SQL database on Microsoft Azure. The focus was on reliability, secure data handling, and keeping things running at scale.

### 2D Game Development in Unity
Built a complete 2D game from scratch during my Bachelor's degree. Gameplay loop, physics interactions, level design — all written in C# using object-oriented patterns. This is where my interest in how software feels to use first came from.
